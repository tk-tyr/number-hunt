# ナンバーハント

答えが 1〜N になる式をすばやく順番にタップするスピードゲーム。  
HTML / CSS / JavaScript のみで動作。スマホ・PCどちらでも遊べる。

---

## 遊び方

1. グリッドサイズと難易度を選んで **スタート**
2. 画面上部の **NEXT** に表示された数が答えになる式を探してタップ
3. 正解 → 緑に光る、不正解 → 赤く震える＋ペナルティタイム加算
4. 全マスタップしたらクリア。タイムとランクが表示される

---

## グリッドサイズ

| サイズ | 問題数 | 答えの範囲 |
|--------|--------|------------|
| 3×3    | 9 問   | 1 〜 9     |
| 4×4    | 16 問  | 1 〜 16    |
| 5×5    | 25 問  | 1 〜 25    |

---

## 難易度

| 難易度 | 内容 | 例（答え = 7） |
|--------|------|---------------|
| **シンプル** | 数字そのまま | `7` |
| **ハード** | 足し算・引き算 | `3＋4`　`12－5` |
| **ベリーハード** | 掛け算・割り算 | `21÷3`　`1×7` |
| **ヘル** | 複合演算・累乗・平方根 | `2×4－1`　`(3＋4)×1`　`√49` |

> **ヘル・ベリーハードの注意**: `×÷` は `＋－` より先に計算（標準の数学記法）。  
> 例: `3×4＋1 = 13`（12+1）、`(2＋3)×2 = 10`（括弧を先に）

---

## ペナルティ

間違えた式をタップするとタイムが加算される。難易度が高いほどペナルティも大きい。

| 難易度 | ミス1回のペナルティ |
|--------|------------------|
| シンプル | ＋1秒 |
| ハード | ＋2秒 |
| ベリーハード | ＋3秒 |
| ヘル | ＋4秒 |

- ペナルティはタイマーに即反映される（タイマーが瞬時に進む）
- ミス時は画面中央に「＋X秒」が大きく表示される
- ゲーム中は画面上部に「累計ペナルティ秒数（ミス回数）」を表示
- クリア後のリザルトでも内訳を確認できる
- ノーミスでクリアすると「ノーミス！」と表示
